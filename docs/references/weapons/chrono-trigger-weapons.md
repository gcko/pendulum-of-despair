    I. Introduction
    ---
    
    This FAQ is a transcription of the Item Encyclopedia found in the Extras
    Menu of Chrono Trigger DS. It contains all the information about each item
    as well as the various other ways to obtain them that the Encyclopedia
    leaves out (you'll notice in the first few versions that there are a lot of
    'etc.'s)
    
    Each section is divided into item type, and they all are found in the same order
    that they appear in the Encyclopedia. If you're looking for a particular item,
    use the CTRL+F feature of your browser to find that specific number (located to
    the left of the item).
    
    And lastly, each item section begin with an example layout of how the section
    will go. Utilize it if you get confused.
