## Quick Start

**Need SFX now?** Open [jsfxr](https://sfxr.me) in your browser and
start generating retro sounds. Free, instant, no account needed.

**Need music now?** Browse [itch.io 16-bit music packs](https://itch.io/game-assets/tag-16-bit/tag-music)
for $5-$20 complete JRPG soundtracks.

**Want AI-generated?** Read [ai-audio-generation.md](ai-audio-generation.md)
for the recommended AIVA + SNES soundfont workflow.
