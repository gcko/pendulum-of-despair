    Introduction
    -------------
    
              Welcome to my Secret Of Mana Game Script FAQ. I have chosen to do a
    Game Script for Secret Of Mana (or SOM from now on) because this was the first
    RPG I ever played and I have really enjoyed this game. I have tried very hard
    to write this script EXACTLY how it appears in the game, but that is hard to
    do, and there may appear to be some typos, but they are like that in the game.
    This document is completely original, I have prepared this script myself from
    start to finish with no help from any other works or sources, except from the
    game itself..
    
    **Note**
     This script contains MAJOR spoilers and probably should not be read by first
    time players of SOM. I suggest that you play through this game at least once
    before reading this FAQ.

    Guide Layout
    -------------
      This guide works as follows:
    
    I have marked "chapters" with a # sign.
    
    A double row of ---- marks are sidequests, optional scenes, story breaks,
    and/or dialogue that does not pertain to the mandatory plot.
    
    Orbs gained are marked with 2*'s.
    
    Minor items to note are marked with a single *.
    
    To find chapters easier, use the FIND tool by pressing CTRL + F.
    
    Feel free to name your characters what you will, I have named my characters as
    follows: Hero, Princess, and Sprite.
    
    For an easier time, people who say the the same things in an area will be
    marked on one line rather than separately, ex.= if Guards 1 and 2 in an area
    both say, "Get out!" then I will mark them as (Guards 1,2: Get out!)
    
    **Note**
    I will not re-cover towns that I have previously visited unless someone says
    something completely different or it is mandatory to the story line.**
