    3. Bestiary		[BTS3]
    ------------------------------
    
    How to use the bestiary in this FAQ:
    
    There are now two ways to search for a particular monster in this
    bestiary.
    
    1) Search by Era or Location. Please see the Table of Contents for
    the 4 letter code for each Era or Location. An easy way for beginners
    to find out where each monster is.
    
    2) Search for each monster by Number. A faster way of finding out
    the monster. Use this if you know which number you're looking for.
    This is for Advance users who already played the game, and want to
    see which monster they have missed in the bestiary. Just enter the
    3 digit number of the monster just to jump to it. Numbers range from
    001 to 246. To find monsters over 100, just add the . after it. This
    is to insure you find the monster right off the bat.
